## Getting started

- Make sure  you have [Node v14](https://nodejs.org/en/download/package-manager) or higher and [Git](https://git-scm.com/downloads) downloaded

```bash
git clone git@github.com:mathdebate09/storika.git
cd storika
npm install
npm run dev
```
